Angular Doge Directive

When your AngularJS application needs a doge.

Example Doge

Doges are scalable with the scale attribute.

0.5 Scale

1.0 Scale

Code Example

<div ng-app="doge-directive" scale="0.5">
    <doge></doge>       
</div>

Install

Install via bower:

bower install angular-doge-directive
